# Hero Video

Configure via `.env`:

- **PUBLIC_HERO_VIDEO_URL** — Any of:
  - YouTube: `https://youtu.be/xxx` or `https://youtube.com/watch?v=xxx`
  - Vimeo: `https://vimeo.com/123456`
  - Direct: `/videos/hero.mp4` or any public mp4/webm URL

- **PUBLIC_HERO_VIDEO_POSTER** — Any public image URL (optional).
  For YouTube, a thumbnail is used automatically if not provided.
